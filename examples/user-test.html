Usero User-Test Plugin Smoke Test

This page mounts the user-test plugin against a local Usero dev server. It activates only when the URL has ?usero_test=<slug>. Without that param it sits idle.

To exercise the plugin:

  1. Boot the SaaS app on http://localhost:5142 and create a UserTest with a known slug (e.g. smoke).
  2. Open this page with the slug, e.g. ?usero_test=smoke.
  3. Allow microphone access when prompted. A red recording bar appears at the bottom.
  4. Speak for ~35s so at least one chunk uploads, then tap Finish.

Network tab should show: 1x POST /api/user-test-sessions, 1+ PUT /api/user-test-sessions/<id>/chunk?index=N, 1x POST /api/user-test-sessions/<id>/finalise.

Reload with usero_test=smoke
